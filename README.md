# avalanche-specs
AI-generated specs from the avalanchego code

- `specs/v0` was generated with Claude Sonnet 4.6 with 200k context with `prompts/generate-specs.md`
- `specs/v1` was generated with Claude Opus 4.8 with 1M context with `prompts/generate-specs.md`
  - Also did a follow-up to deep dive on SAE and Coreth
- `specs-rust` was generated with Claude Opus 4.8 with 1M context with `prompts/generate-rust-specs.md`
  - Went through several follow ups from the initial prompt to deepen and expand parts of the spec.
